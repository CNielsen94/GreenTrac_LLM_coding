# GreenTrac_LLM_coding
This repository holds the data and code for the GreenTrac WP1 paper.

Code definitions used in the experiment is available in the prompts/ directory. These are subject to change.

A report on all disagreements including the provided quotes and reasoning the model provided during the coding. An aggregated summary for each category will be uploaded soon.

IRR_analysis.ipynb shows the results of the latest attempt at the bottom. The raw JSON outputs are available in results_final (named this because of a minor bug in the initial code, causing a double-nested structure that was fixed). The corresponding excel/csv file will be uploaded soon.
